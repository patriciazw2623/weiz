UEFA Euro 2020 Predictions

  • Featured Skills: Machine Learning, Web Development

Seeing it as a multiclass classification problem, I used three machine learning models: Random Forest, Naive Bayes and Convolutional Neural Network. The prediction results are demonstrated in a website built upon the springboot framework.
